# 指南

Zen Platform 是一个声明式全栈应用生成平台。通过在 Ruby Model 中声明 DSL，自动生成管理后台页面和 REST API。

## 核心概念

- **Model DSL** — 在模型中声明字段、关联、展示配置
- **DslTable / DslForm** — 运行时从 DSL 元数据动态渲染列表和表单
- **脚手架生成器** — 一键生成 Controller + 页面 + 路由
- **富文本编辑器** — Lexical 引擎，20 个插件
