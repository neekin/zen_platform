# 内置插件

## 文本格式

| 插件 | 功能 | 快捷键 |
|------|------|--------|
| Bold | 加粗 | Ctrl+B |
| Italic | 斜体 | Ctrl+I |
| Underline | 下划线 | Ctrl+U |
| Strikethrough | 删除线 | Ctrl+Shift+S |
| Code | 行内代码 | Ctrl+E |

## 块级元素

| 插件 | 功能 | 快捷键 |
|------|------|--------|
| Heading | 标题 H1/H2/H3 | — |
| Quote | 引用 | — |
| List | 有序/无序列表 | — |
| Code Block | 代码块（21 种语言） | — |
| Table | 表格（8x8 选择器） | — |

## 富内容

| 插件 | 功能 |
|------|------|
| Image | 图片上传/拖拽/缩放 |
| Link | 链接（Ctrl+K） |
| Video | 视频嵌入（YouTube/Bilibili/Vimeo） |
| Attachment | 附件上传 |
| Mention | @提及 |
| Emoji | 表情符号 |
| Mermaid | 流程图/序列图 |
| Math | 数学公式（KaTeX） |

## 其他

| 插件 | 功能 |
|------|------|
| History | 撤销/重做 |
